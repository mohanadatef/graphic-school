## Features & Functionality

### Can I have multiple instructors per course?

Yes. This is one of our key features. You can assign multiple instructors to each course.

### Can I create multiple groups per course?

Yes. You can create as many groups as needed within each course, allowing you to run parallel classes.

### How does enrollment work?

1. Student submits enrollment request
2. Admin reviews enrollment
3. Admin approves/rejects
4. If approved, student assigned to group
5. Student can access course

### How does attendance work?

Instructors can take attendance for each group session. They mark each student as present, absent, late, or excused. Attendance is tracked and visible to students and administrators.

### Can I customize the public website?

Yes. You have full control over:
- Logo and branding
- Colors and fonts
- Page content (via CMS)
- Multi-language content

### Do you support multiple languages?

Yes. You can add multiple languages, set a default language, and provide content in all languages. RTL languages (like Arabic) are fully supported.
