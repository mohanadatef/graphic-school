## Typography

### Primary Font

**Headings:** Poppins (Bold, Semi-Bold, Regular)
- Modern, clean, professional
- Use for: Headings, titles, CTAs

**Body:** Cairo (Regular, Medium)
- Readable, friendly, approachable
- Use for: Body text, descriptions

### Font Hierarchy

**H1 (Page Title):**
- Font: Poppins Bold
- Size: 2.5rem (40px)
- Line height: 1.2

**H2 (Section Title):**
- Font: Poppins Semi-Bold
- Size: 2rem (32px)
- Line height: 1.3

**H3 (Subsection):**
- Font: Poppins Semi-Bold
- Size: 1.5rem (24px)
- Line height: 1.4

**Body:**
- Font: Cairo Regular
- Size: 1rem (16px)
- Line height: 1.6

**Small Text:**
- Font: Cairo Regular
- Size: 0.875rem (14px)
- Line height: 1.5

### Font Usage

**When to Use Poppins:**
- Headings
- Navigation
- Buttons
- Important labels

**When to Use Cairo:**
- Body text
- Descriptions
- Form labels
- General content
