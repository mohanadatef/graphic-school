# Product Overview

## Introduction

**Graphic School 2.0** is an advanced **Learning Management & Coaching (LMC)** platform designed specifically for training centers and academies. Built with modern technology, it provides a complete, streamlined solution for course management, student enrollment, attendance tracking, and certificate issuance.

## What is Graphic School 2.0?

Graphic School 2.0 is a comprehensive LMC platform that enables educational institutions to:

- **Manage Courses & Groups:** Create courses with multiple instructors, organize students into groups, and schedule sessions
- **Handle Enrollments:** Streamlined enrollment workflow from student registration to admin approval and group assignment
- **Track Attendance:** Manual attendance tracking per session (present/absent/late)
- **Issue Certificates:** Professional certificate generation with QR code verification
- **Engage Students:** Community features for interaction (posts, comments, replies)
- **Customize Branding:** Fully customizable public website with branding, colors, fonts, and multi-language support
- **Manage Content:** Built-in CMS page builder for dynamic, multi-language pages

## Key Features

### 1. Course Management

- **Course Creation:** Full course management with required price, description, image
- **Multiple Instructors:** Assign multiple instructors per course
- **Public Display:** Courses visible on public website with pricing
- **Categories:** Organize courses by categories
- **Multi-Language:** Course content supports translations

### 2. Group Management

- **Multiple Groups:** Create multiple groups per course
- **Group Dates:** Start date and end date for each group
- **Group Notes:** Additional information per group
- **Multiple Instructors:** Assign instructors from course instructors
- **Capacity Management:** Set and track group capacity

### 3. Session Management

- **Flexible Scheduling:** Sessions belong to groups
- **Session Details:** Title, date, time, optional instructor, notes
- **Session Templates:** Optional templates for course organization

### 4. Enrollment Workflow

- **Public Enrollment:** Students can enroll via public website
- **Admin Approval:** Admin reviews and approves enrollments
- **Group Assignment:** Admin assigns students to groups on approval
- **Status Tracking:** Track enrollment status (pending, approved, rejected, withdrawn)

### 5. Attendance Tracking

- **Manual Attendance:** Instructor manually marks attendance (NO QR)
- **Status Options:** Present, Absent, Late
- **Per Session:** Attendance tracked per session
- **History:** Students can view attendance history

### 6. Certificate Management

- **Certificate Issuance:** Admin/Instructor can issue certificates
- **QR Verification:** Each certificate has unique QR code
- **Public Verification:** Anyone can verify certificates publicly
- **Required Fields:** Student, Course, Group, Instructor, Date, Verification Code
- **Branded Templates:** Uses academy branding (logo, colors, fonts)

### 7. Community Features

- **Posts:** Create posts in group communities
- **Comments & Replies:** Full commenting system
- **Likes:** Optional likes on posts/comments
- **Moderation:** Admin moderation tools
- **NO Gamification:** No XP, points, badges, or leaderboards

### 8. Website & CMS

- **Setup Wizard:** First-time setup with guided steps
- **Branding:** Customizable logo, colors, fonts
- **Page Builder:** Built-in CMS page builder
- **Public Pages:** Home, About, Courses, Instructors, FAQ, Contact
- **Multi-Language:** Default English, support for additional languages (Arabic with RTL)

### 9. Multi-Currency Support

- **Default Currency:** EGP (Egyptian Pound)
- **Additional Currencies:** Admin can add more currencies
- **Course Pricing:** Courses priced in selected currency

## Business Model

**Tenancy:** One client = one deployment (own domain, own DB)

This means:
- Each academy gets its own dedicated instance
- Own domain name
- Own database
- Full customization and branding
- Data isolation and security

## Technology Stack

### Backend
- **Framework:** Laravel 11
- **Language:** PHP 8.2+
- **Database:** MySQL 8.0+ / MariaDB 10.6+
- **Authentication:** Laravel Sanctum (Token-based)

### Frontend
- **Framework:** Vue.js 3 (Composition API)
- **Build Tool:** Vite
- **State Management:** Pinia
- **Styling:** Tailwind CSS
- **i18n:** Vue I18n

### Infrastructure
- **Web Server:** Nginx
- **Application Server:** PHP-FPM
- **Cache:** Redis (optional)
- **Queue:** Redis/Database
